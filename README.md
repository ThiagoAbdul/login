### Overview

Uma aplicação com ênfase em simular um login de um site, com cadastro no banco de dados, encriptação de senha, armazenamento de imagens, entre outros recursos.

### Stacks

#### Frontend
* HTML
* CSS
* JavaScript

#### Backend
* Java EE
* Hibernate
* Apache Tomcat 9

#### Banco de dados
* MySQL

#### Outras
* Git
* Maven
